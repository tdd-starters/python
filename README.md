# Python TDD Starter
A simple starter project for TDD katas in Python.

This contains no advanced features for managing dependencies or virtual environments;
it's just two Python files: one for tests and one for the implementation.

Run the test(s) with:
```shell
python -m unittest discover
```

If you want something more advanced,
check out the [Python Project Template](https://github.com/raniz85/python-template) which uses Poetry for managing 
dependencies and virtual environments, and has support for linting and formatting.
